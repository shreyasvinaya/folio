Patra is an open-source, self-hostable platform for publishing documents and static sites, backed by Git and secured with authenticated share links. It ships as a TypeScript monorepo built on Hono, React, and SQLite, with an agent-friendly CLI for automated workflows. Security is a first-class concern: the platform uses scoped tokens, sandboxed content rendering, and SSRF-guarded webhooks, and is verified by 500+ automated tests.
